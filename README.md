赛博禅院是一款赛博朋克和佛经深度结合的window 软件。

不用去寺庙就可以上香，添加贡品。拜佛，修行。为了更好的修行，提供了今日功德和功德排名功能。

何时何地都可以修行，精进佛法。佛学永无止境，佛法无边。
